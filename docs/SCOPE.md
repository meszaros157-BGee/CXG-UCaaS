# Scope

## Pages Included
- Homepage (hero, overview cards, call to action)
- Additional pages TBD based on proposal structure

## Sections Included
- Hero / introduction
- Key pillars or strategic themes (card grid)
- Call to action
- Additional sections as content is finalized

## What Is Excluded
- Backend APIs or server-side logic
- User authentication or login flows
- CMS integration
- E-commerce or transactional features
- Internal tooling or dashboards

## Launch Milestones
1. Scaffold and structure complete
2. Content populated and reviewed
3. Visual polish and responsive QA
4. Stakeholder review
5. Deploy to production URL
